## 設定檔結構

```
configs/
├── companies.yml              # 19家公司 + 上下游關係
│   └── 用於: matcher.py (匹配公司、判斷客戶/供應商)
│
├── topics.yml                 # 主題 + 關鍵字
│   └── 用於: matcher.py (匹配主題)
│
├── sentiment_rules.yml        # 情緒詞典 + 否定詞
│   └── 用於: sentiment.py
│
├── importance_rules.yml       # 重要性評分規則
│   └── 用於: scorer.py
│
├── anomaly_rules.yml          # 異常偵測閾值
│   └── 用於: anomaly.py
│
└── 7d_highlights_rules.yml    # 7日報告亮點規則
    └── 用於: generate_7d_report.py
```

## 關鍵設計決策

| 項目 | 決策 | 原因 |
|------|------|------|
| 不用 LLM | 關鍵字匹配 + 規則 | 成本低、可控、可解釋 |
| 完整內文 | 訪問文章頁抓取 | 主題匹配更準確 |
| 詞邊界匹配 | 短名稱用 `\b` regex | 避免 "ase" 配到 "phase" |
| 基準線最後更新 | Step 7 | 避免今日資料影響今日比較 |
| 每日重生7日報 | 滾動窗口 | 隨時查看「過去7天」 |
| JSONL 格式 | 事件儲存 | 方便追加、grep、串流處理 |
